# 📘 Chapter 2 — Functions and Control Structures (PHP)

Chapter 2 introduces the essential building blocks of PHP programming: **functions**, **conditional logic**, **loops**, and **file inclusion**. These concepts allow you to write organized, reusable, and dynamic code.

---

## 🔹 Functions

This chapter explains how to use functions to structure your code and avoid repetition.

### Key concepts:
- **Defining functions** using the `function` keyword  
- **Calling functions** to reuse code efficiently  
- **Returning values** from functions  
- Understanding **variable scope** (local vs global)  
- Using the **`global` keyword** when a function needs access to external variables  

Functions help create modular, maintainable PHP programs.

---

## 🔹 Making Decisions — Conditional Structures

PHP uses conditional statements to let your program make decisions based on logic.

### Covered topics:
- `if` statements  
- `if...else` statements  
- Nested `if` structures  
- **Compound conditionals** using `&&` (AND) and `||` (OR)  
- `switch` statements  

These tools let your web applications behave differently depending on user input or data values.

---

## 🔹 Repeating Code — Loops

Loops allow PHP programs to repeat actions without writing the same code multiple times.

### Types of loops in this chapter:
- **`while` loops** — repeat while a condition is true  
- **`do...while` loops** — run at least once before checking the condition  
- **`for` loops** — ideal for counting or fixed‑step iterations  
- **`foreach` loops** — best for looping through arrays  

Loops are essential for tasks like processing data sets, generating lists, and building dynamic pages.

---

## 🔹 Including Files

Finally, the chapter introduces file inclusion:

- `include`
- `require`

These allow you to split code across multiple files, making large projects easier to maintain and reuse.

## ⭐ Summary

Chapter 2 provides the core programming tools for controlling the flow of a PHP program:
